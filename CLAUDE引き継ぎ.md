# 塗装レシピ録 / Paint Log v2

ガンプラ・模型の塗装レシピを記録・共有するプラットフォーム。「模型塗装界のクックパッド」を目指す。記録が面倒でない実用ツールであることを最優先する。

## 技術スタック
- HTML / CSS / Vanilla JS（v1の資産を流用）
- Supabase（PostgreSQL + Auth + Storage）
- Cloudflare R2（画像保存・egress無料）
- ホスティング: Cloudflare Pages

## ディレクトリ構造
| パス | 役割 |
|-----|------|
| `index.html` | エントリ |
| `/src` | ページ・機能 |
| `/assets` | 塗料DB・静的データ |
| `v1-reference.html` | v1の参考用（部品取り。直接編集しない） |

## 行動原則
- 3ステップ以上のタスクは必ずPlanモードで開始する
- コードを読まずに書かない。既存コードを確認してから変更する
- 動作を証明できるまで完了としない
- 変更は必要な箇所のみ。影響範囲を最小化する
- 権限設定（Supabase RLS）は最重要。実装したら必ず検証する
- 写真はURLに埋めずR2に保存しURLだけ持つ
- コンテキストが逼迫したら正直に伝え、区切りを提案する

## やらないこと
- いいね/閲覧数ランキング機能（差別化方針）
- スマホアプリ化（Web1本。PWAは可）
- 確定していない仕様の先行実装（必ず確認を取る）
